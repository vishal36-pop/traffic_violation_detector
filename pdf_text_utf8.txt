Course Project: Traffic Rule Violation Detection
AID 728
1 Overview
In this project, you will design a computer vision system to detect traffic rule violations involving
two-wheelers from street images. The goal is to build a robust, efficient, and modular system
under strict resource constraints.
2 Problem Statement
Given a single RGB image captured from a street scene, your system must:
1. Detect all two-wheelers (motorcycles/scooters).
2. Identify whether any of the following violations occur:
•More than two riders on a two-wheeler.
•One or more riders not wearing a helmet.
•Not wearing a helmet and more than two riders.
3. For each violating vehicle, extract and report theregistration number (license plate).
3 Constraints
•Model Size Limit:Total size of all model files must not exceed250 MB.
•No Large Models:Use of large Vision-Language Models (e.g.,>1B parameters) is
prohibited.
•Offline Execution:Internet access will be disabled during evaluation.
•Inference Efficiency:The system should process each image within a reasonable time.
4 Output Format
For each input image, your system must return a JSON-compatible dictionary:
{
"violations": [
{
"num_riders": int,
"helmet_violations": int,
"license_plate": "string"
}
]
}
where:
1
•num riders: Total number of riders detected.
•helmet violations: Number of riders not wearing helmets.
•license plate: Recognized license plate string.
